# Risk Management & Position Sizing Calculator for Scalpers

## Project Overview
Build a fast, instant, and efficient position sizing calculator for options scalpers integrating with Alice Blue broker API.

## Tech Stack
- **Frontend**: React + TypeScript + Vite
- **UI**: TailwindCSS (for rapid styling)
- **State**: React hooks/context
- **API**: Alice Blue option chain REST APIs

## Core Features

### 1. Auto-fetch Underlyings
- Call `getUnderlying` API to populate dropdown with all available symbols (NIFTY, BANKNIFTY, etc.)
- Cache the list to avoid repeated API calls

### 2. Auto-fetch Expiries
- On underlying selection, automatically fetch expiry dates via `getUnderlyingExp`
- Default to nearest expiry

### 3. Auto-fetch Option Chain
- On expiry selection, fetch full option chain data automatically
- Extract: spot price, futures price, lot size, all strike prices with CE/PE data

### 4. Position Sizing Calculator (Main Feature)
**Inputs** (minimal manual entry):
- Trading Capital (₹) - editable, persisted in localStorage
- Risk per Trade (%) - editable, default 1%
- Stop Loss (points) - auto-calculated or manual override
- Option Type (CE/PE)

**Auto-calculated outputs**:
- Max Risk Amount (₹)
- Lot Size (from API)
- Position Size (lots)
- Total Premium (₹)
- Breakeven Price
- Risk/Reward Ratio

### 5. Quick Strike Selection
- Display ATM strike prominently
- One-click to select any strike
- Show premium, OI, and PCR for quick decision

### 6. Risk Visualization
- Show premium vs max risk comparison
- Color-coded risk indicators

## API Integration

```typescript
// Endpoints
const BASE_URL = 'https://ant.aliceblueonline.com/obrest/optionChain';

// 1. Get Underlyings
POST /getUnderlying
Payload: { "exch": "nse_fo" }

// 2. Get Expiries
POST /getUnderlyingExp
Payload: { "underlying": "NIFTY", "exch": "nse_fo" }

// 3. Get Option Chain
POST /getOptionChain
Payload: { "underlying": "NIFTY", "expiry": "13APR26", "interval": "15", "exch": "nse_fo" }
```

## Calculations

```typescript
// Position Sizing Formulas
maxRiskAmount = capital * (riskPercent / 100)
positionLots = Math.floor(maxRiskAmount / (stopLoss * lotSize * multiplier))
totalPremium = ltp * lotSize * positionLots
breakeven = strikePrice + premium (for CE) or strikePrice - premium (for PE)
```
